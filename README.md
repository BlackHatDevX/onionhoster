## ⚡ One Command

```bash
git clone https://github.com/BlackHatDevX/onionhoster.git
cd onionhoster
sudo bash onionhoster.sh
```

That's it. The script detects your OS, installs dependencies, configures Tor, and gives you a menu to:
- Host your `index.html`
- Forward to any app port (Next.js, Flask, etc.)
- Backup/restore your `.onion` address
- Generate new onion addresses

## 🛡️ Security

> OnionHoster is **not** a magic anonymity tool — follow best practices or risk deanonymization.

- Run in a VM or container for isolation
- Don't use the same machine for personal activities
- Use a firewall to prevent clearnet leaks
- Test with Tor Browser to verify your site is only reachable via `.onion`
